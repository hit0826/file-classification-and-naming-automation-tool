# File Classification and Naming Automation Tool

## v1.0.0

- Windows GUI for organizing local files by detailed format, date, type, name initial, or original folder return.
- Content-based renaming for images, readable documents, and code files.
- Original-name restore for files renamed by the content-based tools.
- Safe move behavior with duplicate detection, collision suffixes, hidden/system file skipping, and generated-folder cleanup.
- Standalone Windows executable package for running the GUI without a terminal window.
